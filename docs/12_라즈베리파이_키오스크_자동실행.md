# 12. 라즈베리파이 키오스크 자동실행 (systemd)

`frontend-kiosk/PcbKiosk` 앱을 라즈베리파이 터치스크린에서 부팅 시 자동 실행하는 방법이다.

## 1) 라즈베리파이 패키지 설치

```bash
sudo apt update
sudo apt install -y chromium-browser curl
```

`.NET 8`이 없으면 설치:

```bash
curl -sSL https://dot.net/v1/dotnet-install.sh -o dotnet-install.sh
chmod +x dotnet-install.sh
./dotnet-install.sh --channel 8.0
echo 'export PATH=$PATH:$HOME/.dotnet:$HOME/.dotnet/tools' >> ~/.bashrc
source ~/.bashrc
dotnet --version
```

## 2) 코드 위치 확인

가이드 기준 경로:

- `/home/pi/inspection/frontend-kiosk/PcbKiosk`
- `/home/pi/inspection/frontend-kiosk/deploy/raspberry-pi`

경로가 다르면 아래 서비스 파일의 경로를 같은 방식으로 수정한다.

## 3) 키오스크 앱 빌드

```bash
cd /home/pi/inspection/frontend-kiosk/PcbKiosk
dotnet publish -c Release
```

## 4) 브라우저 실행 스크립트 권한 부여

```bash
chmod +x /home/pi/inspection/frontend-kiosk/deploy/raspberry-pi/start-kiosk-browser.sh
```

## 5) systemd 서비스 등록

```bash
sudo cp /home/pi/inspection/frontend-kiosk/deploy/raspberry-pi/pcb-kiosk-web.service /etc/systemd/system/
sudo cp /home/pi/inspection/frontend-kiosk/deploy/raspberry-pi/pcb-kiosk-browser.service /etc/systemd/system/
sudo systemctl daemon-reload
```

## 6) 자동 시작 활성화 + 실행

```bash
sudo systemctl enable pcb-kiosk-web
sudo systemctl enable pcb-kiosk-browser
sudo systemctl start pcb-kiosk-web
sudo systemctl start pcb-kiosk-browser
```

## 7) 상태/로그 확인

```bash
sudo systemctl status pcb-kiosk-web
sudo systemctl status pcb-kiosk-browser
journalctl -u pcb-kiosk-web -f
journalctl -u pcb-kiosk-browser -f
```

## 8) 유지보수 명령

중지:

```bash
sudo systemctl stop pcb-kiosk-browser
sudo systemctl stop pcb-kiosk-web
```

재시작:

```bash
sudo systemctl restart pcb-kiosk-web
sudo systemctl restart pcb-kiosk-browser
```

비활성화:

```bash
sudo systemctl disable pcb-kiosk-browser
sudo systemctl disable pcb-kiosk-web
```

## 9) 자주 발생하는 문제

- **브라우저가 안 뜰 때**
  - `DISPLAY=:0`이 맞는지 확인
  - 데스크톱 환경(wayland/x11) 로그인 상태 확인
- **앱 연결 대기 반복**
  - `pcb-kiosk-web` 서비스 로그에서 `dotnet` 실행 오류 확인
- **터치 반응 이상**
  - HDMI 터치 USB 케이블 연결 여부 확인

## 10) 한 줄 명령(`kiosk-start`, `kiosk-stop`) 만들기

프로젝트에 포함된 스크립트를 `/usr/local/bin`에 설치하면 어디서든 한 줄로 실행할 수 있다.

```bash
sudo cp /home/pi/inspection/frontend-kiosk/deploy/raspberry-pi/kiosk-start /usr/local/bin/kiosk-start
sudo cp /home/pi/inspection/frontend-kiosk/deploy/raspberry-pi/kiosk-stop /usr/local/bin/kiosk-stop
sudo chmod +x /usr/local/bin/kiosk-start /usr/local/bin/kiosk-stop
```

사용:

```bash
kiosk-start
kiosk-stop
```

이 스크립트는 아래 서비스 중 설치된 것을 자동으로 찾아 실행/중지한다.

- `pcb-kiosk-web.service`
- `pcb-kiosk-browser.service`
- `pcb-react-kiosk-web.service`
- `pcb-react-kiosk-browser.service`
